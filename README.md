# vfox-seaweedfs

[![CI](https://github.com/jbox-web/vfox-seaweedfs/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-seaweedfs/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [seaweedfs](https://github.com/seaweedfs/seaweedfs)

## Install

```
mise plugin install seaweedfs https://github.com/jbox-web/vfox-seaweedfs.git
```
